# Sky Studio Lite — 15半音阶钢琴

一个极简的网页版半音阶钢琴模拟器，从 C4 到 C6 共 15 个音，支持移动端和桌面端触屏操作。

![预览图](https://via.placeholder.com/800x400?text=Sky+Studio+Lite) *(可替换为实际截图)*

## ✨ 特性

- 🎵 **精确音高**：基于十二平均律，从 C4 (261.63Hz) 平滑过渡到 C6 (1046.50Hz)，跨度 24 个半音
- 🎹 **3×5 网格布局**：紧凑简洁，纯黑背景，无多余视觉元素
- 🔢 **简谱标记**：按钮显示简谱风格数字（带点表示八度），直观易读
- 🔊 **纯正弦波发声**：音色干净，无泛音干扰，支持 Web Audio API
- 📱 **触屏优化**：禁用滚动和长按菜单，响应式适配横屏不同尺寸
- ⚡ **零依赖**：纯原生 HTML/CSS/JS，无需任何外部库

## 🛠️ 技术细节

- **音高计算**：`f = 261.625565 × 2^(偏移半音数 / 12)`
- **相邻音程**：24 / 14 ≈ 1.714 半音（均匀分布）
- **音频**：正弦波振荡器 + 增益包络（淡入淡出）
- **响应式**：使用 `vw` 单位和媒体查询适配横屏
